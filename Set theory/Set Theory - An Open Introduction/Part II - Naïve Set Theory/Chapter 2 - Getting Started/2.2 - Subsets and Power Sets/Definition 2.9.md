$(\forall x \in A) \varphi$ abbreviates $\forall x (x \in A \rightarrow \varphi)$.

Similarly, $(\exists x \in A) \varphi$ abbreviates $\exists x (x \in A \land \varphi)$.

- $A \subseteq B$ iff $(\forall x \in A) x \in B$. See [[Proposition 2.8]].
